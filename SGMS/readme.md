# 学生成绩管理系统实践
简单的C语言控制台应用程序综合示例。
## 主要功能
### 数据的添加、编辑和删除
可以实现简单数据的初步处理，基本的添加删除操作，依赖于数组、文件管理、指针。
### 数据的分析与处理
可以实现简单数据的简单分析，求平均，判断及格人数，以及某人是否在及格线内。
同样，也可以根据一些指令生成成绩表(按排名或学号)与成绩单。
今后会支持数据查找功能，可能试着在搜索过程中支持通配符。

## 程序架构
* sgms.c | 主程序
* settings.bin | 设置文件（首次启动创建，后期如文件有效则调用设置，跳过初次设置步骤）
* data.bin | 数据文件（首次启动创建，后期如文件有效则调用数据）
## 使用介绍
# 1.0版本
简单的字符控制，键入数字实现对系统的控制。

模拟控制台命令。

例如：

    >porfile
        错误：未找到命令"porfile"，请检查拼写。
    >addprofile
        请输入学号：350234
        请输入姓名：张三
        请输入成绩：98
        确认无误？
        （键入Y或N以接受或否定）>Y
        已写入。
    >editprofile /num 350234 95
        是否将学生 张三 的成绩从 98 改为 95？
        （键入Y或N以接受或否定）>Y
        已修改。
    >find /num 350234
        |找到以下结果=====|
        |学生 |学号  |成绩|排名|
        |张三 |350234|95 |91  |
    >

若检测到 setting.bin 可用则沿用里面的设定，无需初次设定。

        |==================================================|

    >
# 命令大全
## 操作命令
* 退出

        exit
* 帮助

        help
        help <具体命令> 查看该命令的帮助
* 添加数据

        addprofile <学号> <姓名> <分数>
        addprofile           //弹出输入菜单
* 修改分数

        editprofile /num <正确的学号> <修改后的分数>
        editprofile /name <正确的姓名> <修改后的分数>
* 删除数据

        delete /name <正确的名字>
        delete /num <正确的学号>
* 查找（非模糊查找生成成绩单）

        find /num <学号>
        find /name <姓名>
* 设置
        
        set                 //弹出设置菜单
        set <内部命令>       //直接调用set内部命令，无需调用菜单
    内部命令：

        size                //查看样本量
        averange            //查看平均分
        total /num          //查看排序好的成绩表（按学号）
        total /score        //查看排序好的成绩表（按成绩）
        ispassed            //查看及格人数和及格率
